USAGE OF TESTOSD: What are the files associated with TestOsd and where are they present? 1. TestOsd.gpr :- \App_Projects\Eval_Src\OSD\TVOsd 2. SliderCharSet1Bpp.gfn:- \App_Projects\Eval_Src\OSD\TVOsd 3. ArialFont.gfn:- \App_Projects\Eval_Src\OSD\TVOsd - (This file is also used by the main HudsonTVOsd.gpr ) 4. Brightness.bmp: - \App_Projects\Eval_Src\OSD\TVOsd\Bitmaps 5. Contrast.bmp:- \App_Projects\Eval_Src\OSD\TVOsd\Bitmaps 6. color.bmp:- \App_Projects\Eval_Src\OSD\TVOsd\Bitmaps 7. GenFilesTestOsd: \App_Projects\Eval_Src\OSD How to Use TestOsd? 1. Open the TestOsd.gpr with WorkBench4 and above. 2. Build the Workbench project using the toolbar option within Genesis WB Tool. 3. After Step2, Generated files for TestOsd will appear in GenFilesTestOsd folder. 4. Copy these Generated files into GenFilesPAL_Asia, GenFilesPAL_Euro and GenFilesNTSC folders. 5. Now build the hudson2_proj.ide using Paradigm. 6.Ensure that there are no compilation errors/warnings. 6. Use the hex file to verify the OSD on the display unit. 7. Keys that can be used with the TestOsd from the remote are, Power, Up, Down, RightArrow, LeftArrow, Menu and Exit.
