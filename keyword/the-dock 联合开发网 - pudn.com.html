# TheDock - Genesis Starter Child Theme [![TheDock Genesis Starter Theme v1.2.4 badge][changelog-badge]][changelog] [![Version 1.2.4 Badge][version-badge]][changelog] [![Last commit badge][last-commit]][last-commit-link] ![TheDock Screenshot](https://github.com/LuisColome/the-dock/blob/master/screenshot.png) ## Author: Luis Colom ( [@luiscolome](https://github.com/LuisColome/the-dock/blob/master/https://twitter.com/luiscolome) / [luiscolome.com](https://github.com/LuisColome/the-dock/blob/master/https://luiscolome.com) ) [Demo](https://github.com/LuisColome/the-dock/blob/master/https://demo.luiscolome.com/thedock/) ## Summary WordPress Starter Theme for use as Child Theme of the Genesis Framework for building custom themes. Uses SCSS and PrePros to compile it. Tested up to WordPress 5.8.1 and Genesis 3.3.5. ## Usage It is set up to use with [Prepros](https://github.com/LuisColome/the-dock/blob/master/https://prepros.io/) to complie the SCSS (with source maps). Javascript is minify (with source maps) Rename folder to your theme name, change the `style.css` intro block to your theme information. Code as you will. Please, keep in mind it is a working in progress project. I'm using the Theme in every project, so I would be changing, improving or even remove stuff in the future. Thank you :octocat: ### Features 1. SCSS with Prepros to compile it. 2. Easy to customize 3. Child theme tweaks 4. WordPress cleanup functions 5.
